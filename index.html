About P2PSP (Peer-to-Peer "Straightforward" Protocol)

The P2PSP is an application layer protocol for the real-time streaming of multimedia content over the Internet, i.e., where the users playback the stream in a synchronized way. It can be used to build a variety of live streaming services that ranges from small hangouts to large IPTV systems.

Learn more about it here at P2PSP protocol.

About the project

In this project, my task was to port the P2PSP client as an Android application. My project aims at doing what a p2psp-console does, but on an Android based device. I was given a task of implementing the P2PSP protocol thus making an Android powered device an "android-peer".

It ressembles the part 3 of the process as depicted in the image, i.e recieving the chunk sent to the android-peer, which instead sends its chunks to other android-peers so that all the peers have the whole stream.

TASKS COMPLETED

PRE-BUILD TASKS

Porting Boost to Android: Boost dependencies related to building p2psp dbs files were successfully resolved by using Prebuilt Boost Libraries. The linking was done using gradle syntax structure.

Editing core files: I had used the primitive version of prebuilt Boost libraries for Android which was short of some modules including the log module and hence as a workaround I had to edit the TRACE messages which were used to display the message to the console, which otherwise would have been throwing errors during the build.

POST-BUILD TASKS

Implementing LibVLC on Android: LibVLC for Android is aimed at enabling the Android devices with the powers VLC player and hence fulfilling our player for android-peer

TASKS PENDING (Work in progress)

Implementing LibVLC on Android: Building LibVLC for Android is being worked upon and issues related to it's build specifically the error related to "vpx" is yet to be resolved.

Testing the Android Peer: Testing the android-peer is still under progress. I am presently working on resolving the issue related to the native jni method implementation.

FUTURE PROSPECTS

ABI Support: Current project version builds p2psp shared library for three specific ABI's namely the armeabi, armeabi-v7a, x86 only. Project build support for all ABI's will be one of the improvements I'll be working upon.

Upgrading Boost dependencyThe project as of now uses prebuilt Boost Libraries for Android, the Boost version being ver 1.53. Upgrading the Boost dependencies, that is, the static libraries on which the core P2PSP files have been built, to the latest version and specific to all Android compatible ABI's, will be deployed in the future.
